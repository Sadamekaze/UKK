Dasar-dasar CSS: Menghias Halaman Web

Dipublikasikan pada 12 Juni 2025 oleh Tim PPLG

Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendeskripsikan tampilan dan format sebuah dokumen yang ditulis dalam bahasa markah seperti HTML. Jika HTML adalah kerangka sebuah rumah, maka CSS adalah cat, perabotan, dan dekorasinya. CSS memungkinkan Anda untuk mengontrol warna, font, tata letak, dan banyak aspek visual lainnya.

Sintaks Dasar CSS

Aturan CSS terdiri dari sebuah selektor dan sebuah blok deklarasi. Selektor menunjuk ke elemen HTML yang ingin Anda beri gaya. Blok deklarasi berisi satu atau lebih deklarasi yang dipisahkan oleh titik koma, di mana setiap deklarasi terdiri dari properti dan nilai.

  • Selektor: Menargetkan elemen (contoh: h1, .intro, #header).
  • Properti: Aspek yang ingin diubah (contoh: color, font-size).
  • Nilai: Pengaturan untuk properti tersebut (contoh: blue, 16px).

Contoh Kode

Berikut adalah cara memberi gaya pada judul dan paragraf menggunakan CSS internal: 

<!DOCTYPE html>
<html>
<head>
  <title>Styling dengan CSS</title>
  <style>
    h1 {
      color: hsl(var(--p)); /* Warna primer dari tema */
      text-align: center;
    }
    p {
      font-family: sans-serif;
      font-size: 18px;
    }
  </style>
</head>
<body>
  <h1>Halaman Ini Keren!</h1>
  <p>Paragraf ini terlihat lebih rapi.</p>
</body>
</html>
